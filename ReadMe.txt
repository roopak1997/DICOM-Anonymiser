This application is a DICOM Anonymiser that can be used to remove burnt-in ext on an DICOM as well as modify MetaData Fields.
The application is still under development.

Best un on ubuntu with PyCharm

East detector can be found at - https://drive.google.com/file/d/1L16wtZFNxGv885J1VlSbcJAjlIGdS5A2/view?usp=sharing
Place it in ./Resources
